TimeROI - Productivity Tracking App

TimeROI is a mobile app I built with React Native to help track how time is spent throughout the day and encourage better productivity habits. 
The idea is to make it easier to see where your time goes and stay consistent with things you care about.

Right now, the app lets users create an account and log different activities. It keeps track of daily streaks to encourage consistency, and there’s a badge system for hitting certain milestones. 
I also added a focus timer to help users with staying on task during work sessions.

I built this mostly to explore how apps can influence user behavior, especially with things like streaks and rewards. 
A lot of the work went into figuring out how to structure the app, manage user data, and design features that people would might actually want to use regularly.

Tech-wise, it’s built using React Native, and simple databases for now, the backend is not completely finished yet.

Some things I’d like to add next are better data insights (like showing how time is spent over time) and some kind of smart suggestions to help users improve their habits using AI.
